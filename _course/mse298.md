##### Description

* * *


This class focuses on all kinds of engineering communication, ie. memos, proposals and more! You also learn to use Solidworks, which is a useful CAD tool.   

##### Thematic Questions

* * *


How do engineers communicate? 
What considerations go into a materials selection project and how do you back these up? 
How can engineers leverage solidworks?
